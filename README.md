# 《Pro Git（第 2 版）》电子书

**声明**：这个仓库只用于制作电子书。《Pro Git（第 2 版）》一书由原翻译团队翻译，我只是把他们的译稿拿来制作电子书。

## 电子书下载

- PDF 格式在 [`builds/pdf`](/builds/pdf/) 文件夹中

## 制作工具

这个仓库使用驱动 [self.pub](http://self.pub) 的工具链制作电子书。
